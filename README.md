# 💼 Wallet App CLI

A simple TypeScript-based CLI app for managing wallets.

---

## ⚙️ Setup

Clone the repo and install dependencies:

```bash
git clone git@github.com:neha-saggam/wallet-app.git
cd wallet-app
yarn install
```

## ⚙️ Build the Project
To compile the TypeScript source code to JavaScript, run:

```bash
yarn build
```

## 🚀 To Run the App
```bash
yarn start
```

## 🧪 To Test the App
```bash
yarn test
